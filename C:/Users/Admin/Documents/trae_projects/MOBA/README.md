# Heroes of War - MOBA Game

A comprehensive Unity MOBA game project featuring:

## Features

- **Complete MOBA Systems**
  - Hero selection and customization
  - Battle management and gameplay
  - Team matchmaking
  - Match history and statistics

- **Social Features**
  - Friends system
  - Clan/guild management
  - Chat system
  - Voice chat

- **Game Progression**
  - Rank system
  - Battle pass
  - Achievements and badges
  - Hero mastery

- **In-Game Economy**
  - Shop system
  - Inventory management
  - Skin customization
  - Rune system

- **Advanced Features**
  - Replay system
  - Spectator mode
  - Tournament system
  - AI opponents

## Setup

1. Open Unity Hub
2. Add project from file
3. Open `HeroesOfWar` project
4. Let Unity import all assets
5. Press Play to run the game
